# Portfolio

My personal portfolio website showcasing my work as a Web Developer, System Security Analyst, and Creative Technologist.

Built with **React**, **Vite**, **Framer Motion** for animations, and modern CSS for a dark neon theme.

### Features
- Responsive design (desktop + mobile)
- Theme toggle (dark/light)
- Smooth scrolling and animations
- Projects showcase with live links
- LinkedIn badge integration
- Contact form (via EmailJS)

### Live Demo
https://oluwayomi-ogunniyi.github.io/Portfolio/

### Tech Stack
- React + Vite
- Framer Motion
- EmailJS
- Custom CSS with neon gradients and glow effects
- GitHub Pages deployment
